# HobbyTrack

HobbyTrack is a calm practice tracker for hobbies and skills people actually want to improve. It helps users plan their week, log focused practice sessions, see which hobbies are getting attention, and build consistency without turning creative progress into admin.

The current version is a lightweight private-beta MVP built as a static frontend application with local browser storage.

## Product Positioning

HobbyTrack is designed for hobbyists, makers, lifelong learners, and people with real weekly schedules who want a clearer way to keep showing up.

Instead of gamifying hobbies with pressure-heavy streaks, HobbyTrack focuses on:

- realistic weekly practice targets
- simple session logging
- visible progress over time
- next-step clarity after every session
- a calmer relationship with consistency

## Current MVP Features

- Start-free signup flow for a local MVP workspace
- First-run onboarding for setting up one hobby or skill
- Weekly practice target selection
- First next-step planning
- Dashboard with today's focus, weekly progress, streak, total time, active hobbies, milestones, and timeline
- Add-hobby flow with Free plan limit messaging
- Session logging with date, duration, focus, note, next step, mood or energy, and optional milestone
- Hobby detail view with weekly progress and recent sessions
- Weekly review summary
- Local persistence with `localStorage`
- Early beta feedback and join-beta email actions
- Responsive layout for desktop and mobile

## Current Limitations

- Data is stored locally in the browser only
- No real authentication or user accounts
- No cloud sync across devices
- No backend database
- No payment processing
- Pro and Early Supporter plans are planned but not active
- Email links are placeholders for private-beta feedback and access requests
- Export and reminders are not implemented yet
